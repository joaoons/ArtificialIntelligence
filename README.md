# ArtificialIntelligence

This repository contains a Python program to solve **NURUOMINO** puzzles as part of the Artificial Intelligence course.

## Collaborators
This project was completed in collaboration with:
- Edmundo Santos

## Project Objective

Develop a Python program that, given an instance of NURUOMINO, returns the **unique solution**: all regions contain a piece according to the puzzle rules.

- Input: NURUOMINO instance via **standard input** (format defined in Section 4.1 of the assignment)  
- Output: Solution printed to **standard output** (format defined in Section 4.2 of the assignment)
